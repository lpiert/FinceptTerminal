## 🎯 关键概念速查

### [FREE-MODE-STUB] 标记
**含义:** 标识为免费模式而修改的代码位置  
**格式:** `// [FREE-MODE-STUB] 说明文字`  
**作用:** 
- 快速定位所有修改点
- 便于恢复到商业版
- 代码审查时重点关注

**示例:**
```cpp
// [FREE-MODE-STUB] Always return true to unlock all features
bool has_paid_plan() const {
    return true;
}
```

---

### 打桩预留
**含义:** 保留付费功能的接口,暂不实现  
**目的:** 
- 后续可快速启用
- 不影响现有架构
- 便于A/B测试

**示例:**
```cpp
void FinceptMacroPanel::on_fetch() {
    // TODO: [FREE-MODE] Connect to free_macro_aggregator.py
    // Current implementation shows "Coming Soon"
}
```

---

### 无限信用点
**实现:** 设置为 `999999`  
**显示:** UI层特殊处理为 `∞` 或 `UNLIMITED`  
**理由:** 
- 避免大量条件判断
- 保持数据结构兼容
- 易于识别免费模式
